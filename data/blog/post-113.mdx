Codex 和 Claude Code 我都是深度用户，消耗了海量 Token下面把这段时间的一些体感记录下来

## 一、先说 Codex

### 产品形态:我最常用的是桌面端 App

Codex 目前的产品形态有三种:CLI、桌面端 App 和云端。我个人使用最多的是**桌面端 App**。

桌面端的优势非常明显:

- **左侧菜单栏对多项目并行开发非常友好**,可以同时挂好几个项目,来回切换
- 内置 Chrome 控制和 Computer Use,一些简单的每天重复操作类工作可以交给它
- 相比 TUI, GUI 阅读起来更舒服,长时间盯着不容易累

### 性价比 OK,但上下文是硬伤

Codex 这个模型目前来说还是相对有性价比的。但最新的 5.5 **只有 256K 上下文**,这是一个非常大的痛点——稍微大一点的项目，复杂一点的需求，它都会频繁触发上下文压缩

### 它擅长什么,不擅长什么

**不擅长前端。** Codex 在前端开发方面,即使叠加各种 skills,效果依然不如 Claude。所以任何 UI 相关的需求,我都会**无脑交给 Claude **去做,这一点几乎不用犹豫。

**擅长逻辑推理。** Codex 最擅长的是逻辑推理类的工作。在业务流程规范清晰、逻辑链条明确的场景下,它做得又快又好,这是它最值得依赖的地方。

### 几个让人头疼的"坏习惯"

#### 1. 喜欢猜

当然 Claude 也有这个毛病,但 Codex 表现更明显。一般我必须要在 `AGENTS.md` 里加上一条"不要凭空捏造事实",但效果依然有限。

所以一个很重要的经验是:**要给 Codex 提供足够好、足够有效的上下文** 上下文喂得不到位,它就会自己脑补。

#### 2. 特别喜欢写兜底逻辑

Codex 特别擅长、也特别喜欢写兜底逻辑。一旦遇到它解决不了的问题,它不会停下来问你,而是直接给你写一堆 fallback,把问题"盖住"。结果就是代码看起来跑通了,但里面塞了一大堆其实不会被正常触发的兜底分支。导致代码没法看

我的应对方式是在 prompt 里固定加一句:

> 遇到解决不了的问题,或者上下文缺失的时候,停下来咨询我,不要自己往下写,不要写兜底逻辑掩盖问题。

这一句话能省掉很多后续 review 的麻烦。

#### 3. 话不太好懂

大家经常吐槽的"Codex 说的话听不懂"这个问题,在最新版本里依然存在。顺带一提,**Claude 最近也开始有点这个倾向了**,虽然没那么严重。

## 二、再说 Claude

### 第一印象:贵,但是真强

Claude 给我的第一个体感就是——**贵**。

但最新的 **Opus 4.7** 确实强,尤其是在两个方向上是独一档的存在:

- **架构设计**:能看到全局,能权衡取舍
- **前端 UI 与交互设计**:目前没有对手

只要是涉及"怎么搭"、"怎么拆"、"界面长什么样"的问题,我基本都交给 Claude。这里能明显感觉到 Claude 的有更广的知识和世界观

### 产品形态:我用 CLI 最多

Claude 同样有云端、App 端和 CLI 端三种形态,但我个人使用 **CLI 更多**。

一部分是历史原因,用习惯了。我也用过 Web 端,但坦白说 Web 端的交互体验还是比较一般。

另一个非常重要的原因是——

### Claude Code CLI 是我的"万能 Agent 载体"

Claude Code CLI 对我来说不只是 Claude 的入口,它实际上是**我接入各种其他大模型的载体**,是一个万能 Agent 壳。

我会把像 **Kimi、智谱**这类大模型,通过环境变量的方式接进 Claude Code 里使用。这样一来:

- 工具链和工作流是统一的
- 各家模型可以根据任务灵活切换
- 不用为每个模型重新搭一套调用环境

光这一点,就让 Claude Code CLI 在我的日常工作里很难被替代。

## 三、关于"把长目标交给 AI"这件事

最后想聊一个最近被讨论很多的话题:把长目标交给 AI,让 AI 自己一路做下去。

**做 Demo 的时候,这条路看起来很酷。** 一个 prompt,几分钟出一个能跑的东西,演示效果极佳。

**但要做一个生产级项目,这条路目前还不太靠谱。**

原因其实很朴素:当你自己已经不清楚这个项目**最基本的实现原理**的时候,你根本没办法给 AI 一个好的提示词,也给不出精准的上下文。那么 AI 做出来的东西,和你心里想要的东西出现偏差,几乎是必然的——而且偏差还会随着项目时间线越拉越大。

工具再强,前提始终是:**你自己得知道你要什么,以及它大概应该长什么样。**

AI 是放大器,不是替代品。脑子里有清晰的图,它能帮你画得又快又好;你脑子里是糊的,它只会把这层糊放大十倍交回来。
